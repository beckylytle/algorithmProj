Welcome to my mini algorithm visualization project!

During thanksgiving break 2k16, I wanted to make a lil something that would reflect my new knowledge of various algorithms (thanks CS 340!) paired with my past knowledge of D3.js.

I thought I'd whip up a ~game~ in which a random, connected graph is generated. Then, the user has to pick the shortest path in this graph between two designated nodes. On the back end, I coded Dijkstra's algorithm so the user would be able to see if the path they chose was truly the shortest.
